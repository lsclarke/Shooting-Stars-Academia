# Shooting-Stars-Academia

Combining entertainment and education to further growth and development. This educational game project with multiple subjects to learn from and encourage young students to learn in an interactive way and apply the skills they have learned in a game setting. The student learning goals for this project are listed below based on each game mode in the Academia software.

- Game Modes and Learning Subjects:

- Star Boy and the Monochromatics

  Game Objective:
    - Bring color and transparency back to the Brightville Islands.
    - Defeat the evil Mono and his gang known as the Chromatics. Each of whom have stolen a sacred color of the world.

  Narrative Helper:
   - Chroma

  Bosses:
    - Paprika = RED.
    - Oregano = Green.
    - Lapis = Blue.
    - Mono = Alpha and Saturation.
   
  Student Learning Objectives:
    - Students understand the color wheel and know how to define primary, secondary and tertiary colours.
    - Students understand the color wheel through practical mixing of colors.
    - Students understand and can explain the different color schemes Monochromatic, Complementary , Analogous , and Triadic.
    
- Star Boy and the Book of Morse & Alpha

  Game Objective:
    - Find your way out of the Chamber of Code and return back home.
    - Complete each challenge and puzzle to progress further into the chamber and reach the exit.

  Narrative Helper:
   - Professor Betty Alpha

  Bosses:
    - Nuon = noun.
    - Prone = pronoun.
    - Verb = verb.
    - Professor Cody Morse = adjective.
   
  Student Learning Objectives:
    - Students understand the alphabetical order, and is able to determine its morse code counter part 
    - Students understand the alphabets and is able to spell out words, and fill in the blanks.
    - Students understand the morse code variation of a particular word on display and read it back.
